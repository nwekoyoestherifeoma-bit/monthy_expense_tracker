# Monthly Expense Tracker 

A simple, interactive command-line application built to help users track their monthly spending, analyze their budget, and visualize their expenses.

## Features
* **Interactive Data Entry:** Prompts users to input expense names and amounts dynamically.
* **Data Analytics:** Uses Pandas and NumPy to instantly calculate total spending, average expense, highest/lowest expenses, and remaining balance.
* **Data Visualization:** Generates a clean bar chart using Matplotlib to show a breakdown of expenses at a glance.

## Technologies Used
* **Python 3**
* **Pandas:** For structuring expense data into dataframes.
* **NumPy:** For performing quick statistical calculations (mean, sum, max, min).
* **Matplotlib:** For rendering the visual expense bar chart.
* **Google Colab:** The interactive notebook environment used for development.

##  How It Works
1. Run the notebook cells sequentially.
2. Enter your total monthly earnings when prompted.
3. Input your expenses one by one (e.g., Rent, Food, Savings).
4. Type `no` when you are done adding expenses.
5. View your generated **Expense Report** and data visualization chart instantly
